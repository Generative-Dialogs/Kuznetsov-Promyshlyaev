Подробное описание

Класс для генерации изображений с использованием Google Gemini API.

Реализует интерфейс ImageGeneratorProtocol для генерации изображений. Использует Google Gemini API для создания изображений на основе текстовых промптов и опциональных изображений персонажей.

Заметки
Требуется прокси/vpn для работы с Google API
