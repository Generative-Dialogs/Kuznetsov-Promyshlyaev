Класс для распознавания речи (Speech-\/to-\/\+Text) 

Реализует паттерн Singleton для обеспечения единой точки доступа к функционалу распознавания речи. Использует библиотеку speech\+\_\+recognition для преобразования аудио в текст.

\begin{DoxyNote}{Заметки}
Поддерживает русский и английский языки для распознавания 
\end{DoxyNote}
